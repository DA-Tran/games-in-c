# C Terminal Games Suite

## Overview
Collection of 8+ classic games in pure C using stdio/ANSI escapes for text visuals (no external libs). Visual menu, ASCII art/grids, colors, audio (Beep for piano).

**Games:**
- Pacman
- Hangman
- Rock Paper Scissors
- Snakes and Ladders
- Dino Game
- Virtual Piano (visual keys + beeps)
- Snake Game
- 2048
- + extras (Blackjack, Minesweeper, etc.)
